# 🤖 AI Interview Agent

> An AI-powered interview preparation platform built to help students and job seekers practice interviews, improve technical skills, and prepare confidently for real-world placements.


## 📖 About the Project

AI Interview Agent is a full-stack web application that simulates an interview preparation environment using modern web technologies and AI-driven features. The platform enables users to practice interview questions, manage their profiles, upload resumes, track their progress, and improve their interview performance through an intuitive and responsive interface.

The main objective of this project is to provide a smart, user-friendly, and interactive platform that makes interview preparation more effective and accessible.


## ✨ Features

* 🔐 Secure User Authentication & Authorization
* 👤 User Profile Management
* 🤖 AI-Powered Interview Practice
* 📄 Resume Upload & PDF Processing
* 📊 Performance Dashboard & Analytics
* 📈 Progress Tracking

## 🚀 Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Coder-Utkarsh72/AI-Interview-Agent.git
```

### 2️⃣ Install Frontend Dependencies

```bash
cd client
npm install
npm run dev
```

### 3️⃣ Install Backend Dependencies

```bash
cd ../server
npm install
npm run dev
```



## ⚙️ Environment Variables

Create a `.env` file inside the **server** directory.

```env
PORT=

MONGODB_URI=

JWT_SECRET=

RAZORPAY_KEY_ID=

RAZORPAY_KEY_SECRET=

FIREBASE_API_KEY=

FIREBASE_AUTH_DOMAIN=

FIREBASE_PROJECT_ID=
```
